# Intentify

Intentify is a lightweight web tool that helps decode the real meaning behind messages, emails, and dating bios.

It analyzes text and detects:
- Tone
- Interest level
- Emotional signals
- Directness
- Clarity
- Potential red flags

It also suggests possible reply styles.

## Live Demo

https://thedarkduke33.github.io/intentify/

## Features

- Single message analysis
- Compare two messages
- Dating profile analyzer
- Tone detection
- Red flag detection
- Suggested replies
- Copy/share results

## How it works

Intentify uses a rule-based analysis engine that runs entirely in the browser.  
No external API is required, and no user data is stored.

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages hosting

## Disclaimer

Intentify provides interpretation based on language patterns.  
It is not psychological or professional advice.
